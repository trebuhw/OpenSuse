dotfiles zarządzane przez gnu stow

sudo pacman -S stow

1. cd ~/Dokumenty/GitHub/

2. git clone --depth=1 https://github.com/trebuhw/hyprland.git

3. cp ~/Dokumenty/GitHub/hyprland/archlinux/.dotfiles/ ~/

4. Po wykonaniu jakichkolwiek zmian zrobić: rsync -rlpv ~/.dotfiles/ ~/Dokumenty/GitHub/hyprland/archlinux/

5. Wypchnięcie zmian do git

- cd ~/Dokumenty/GitHub/hyprland
- mgs
- mgcom
- mgpush
